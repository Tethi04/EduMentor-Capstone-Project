# EduMentor Capstone Project

An AI-powered educational assistant system with multi-agent architecture.

## Project Structure
- `main.py` - Main application entry point
- `agents.py` - Contains AI agent definitions
- `tools.py` - Utility functions and tools
- `requirements.txt` - Dependencies

## Features
- Multi-agent AI system
- Educational content generation
- Interactive learning tools
- Real-time assistance

## Installation
```bash
git clone https://github.com/Tethi04/EduMentor-Capstone-Project.git
cd EduMentor-Capstone-Project
pip install -r requirements.txt
